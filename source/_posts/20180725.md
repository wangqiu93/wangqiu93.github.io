#### MAC SSH Key
1. 生成：ssh-keygen -t rsa -C "wangqiu8077@163.com"
2. 复制：pbcopy < ~/.ssh/id_rsa.pub
3. 验证：ssh -T git@github.com

#### 版本回退
1. git log ：找到目标版本的id
2. git reset \--hard id ：本地回退到目标版本
3. git push origin HEAD \--force ：推送到远程仓库
