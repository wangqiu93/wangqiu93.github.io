### Git

#### MAC SSH Key
一台电脑可以配置多个ssh key，方便push到不同的代码平台。

1. 生成：ssh-keygen -t rsa -C "代码平台上的邮箱"
2. 复制：pbcopy < ~/.ssh/id_rsa.pub
3. 验证：ssh -T git@github.com

#### 版本回退
1. 找到目标版本的id ：git log
2. 本地回退到目标版本 : git reset \--hard id
3. 推送到远程仓库 ：git push origin HEAD \--force
